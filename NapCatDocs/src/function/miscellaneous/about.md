# 关于本程序

麦麦核心启动器控制台

版本：V4.1.2-beta
新增亮点：
  - 模块化部署逻辑
  - 精确的资源监控器
  - 丰富的可自定义UI界面（rich）
  - 改进的错误处理
  - 实例多开端口自动分配
  - 代理功能

技术栈：
  - Python 3.12.8
  - structlog - 结构化日志
  - rich - 终端UI
  - toml - 配置管理

开源许可：Apache License 2.0
