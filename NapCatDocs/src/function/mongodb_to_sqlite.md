# 数据库迁移

绯雾功能

这个原本是把`0.7.0-`版本的数据库（MongoDB）迁移至`0.7.0+`版本的数据库（SQLite）的，但是因为更高版本的麦麦不再提供迁移脚本，所以这个功能被废弃了。如果迁移目标是比较低的版本，如`0.7.1`、`0.7.2`、`0.8.1`等仍然可以使用，但高版本如`0.11.6`就无法使用了。（不会吧不会吧，不会还有人迭代这么多次了还在用破甲的0.6.2吧？虽然我必须承认那是最好用的破甲版本[超小声bb]）
