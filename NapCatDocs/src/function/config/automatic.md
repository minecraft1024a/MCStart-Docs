# 自动检索实例

进入方式：

主菜单`[B] 🔧 配置管理（新建/修改/检查配置）` → `[A] ✨ 自动检索`

选择后，按照提示分别输入“新的配置集名称”、“要检索的Bot类型，通过输入`A`或`B`选择 MaiBot 或 MoFox_bot ”，以及很重要的<mark>检索的基目录</mark>。

你要确保你的实例在这个目录下，程序会自动检索这个目录下三级的深度，以`bot.pt`所在的目录为主程序根目录，以`main.py`所在的目录为适配器的根目录（如有）。

目录检索完成后，会自动解析`<主程序根目录>/config/bot_config.toml`中的`nickname`字段值，并询问是否将其作为新配置集的昵称。

昵称配置完成后，将询问你是否将`qq_account`字段的值作为当前配置集的QQ号。

QQ号配置完后，将要求你配置序列号、版本号，询问你是否安装了适配器、NapCat、MongoDB、WebUI，并一一检测，如果没有检测到，会要求你手动输入。

全部配置完成后，将会自动录入配置集的保存文件，此时返回主菜单，你就可以通过选项[**<u>`[A] 运行实例`</u>**](../start.md)来启动这个实例了。
