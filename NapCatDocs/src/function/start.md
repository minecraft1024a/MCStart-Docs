# 运行实例/实例多开

## 运行实例
1. 在程序主菜单中选择选项`[A] 运行实例`，你将看到类似这样的界面（前提是你得事先配置了实例）：
    ```
    请选择您要使用的实例：
    ┏━━━━━━━┳━━━━━━━━━━━━━━━━━━┳━━━━━━━━━━┳━━━━━━━┳━━━━━━━━━━━┳━━━━━━━━━━━┓
    ┃ 序号  ┃ 序列号            ┃ 昵称     ┃ 版本  ┃ Bot类型   ┃ 状态      ┃
    ┡━━━━━━━╇━━━━━━━━━━━━━━━━━━╇━━━━━━━━━━╇━━━━━━━╇━━━━━━━━━━━╇━━━━━━━━━━━┩
    │ 实例1 │ 1 (绝对: 1)      │ 默认配置 │ 0.0.0 │ MaiBot    │ ❌ 未配置 │
    │ 实例2 │ mofox (绝对: 2)  │ 墨狐     │ main  │ MoFox_bot │ ✅ 已配置 │
    │ 实例3 │ maimai (绝对: 3) │ maimai   │ main  │ MaiBot    │ ✅ 已配置 │
    │ 实例4 │ mm (绝对: 4)     │ mm       │ main  │ MaiBot    │ ✅ 已配置 │
    │ 实例5 │ mom (绝对: 5)    │ mom      │ dev   │ MoFox_bot │ ✅ 已配置 │
    └───────┴──────────────────┴──────────┴───────┴───────────┴───────────┘
    请输入您要使用的实例序列号（输入Q返回）： ():
    ```
2. 根据提示，输入你想启动的实例的<mark>序列号</mark>，然后回车。
3. 根据实例类型的不同，程序将<mark>给出不同的预设启动项</mark>。
4. 选择你心仪的预设启动项，或者选择`[H] 高级启动项`，你将看到类似这样的界面：
    ```
    [🛠️ 高级启动项]
    ==================================================
    可多选，请使用英文逗号','分隔选项（例如: 1,3）
     [1] 主程序 - ✅ 可用
     [2] 适配器 - ✅ 可用
     [3] NapCatQQ - ❌ 未配置
     [4] 检查MongoDB - ❌ 未配置
     [5] 控制面板 - ❌ 未配置
     [Q] 返回
    请选择要启动的组件:  ():
    ```
    根据提示，输入你想启动的组件的<mark>**序号**，可以一次性启动多个组件，要用**英文逗号**分割</mark>，然后回车启动。
## 实例多开
在<mark>启动首个实例后</mark>，程序主程序的选项`[A] 运行实例`将被替换为`[A] 实例多开`。此时你可以通过这个选项启动多个实例，并同时运行。
1. 在程序主菜单中选择选项`[A] 实例多开` → `[A] 创建新的多开实例` → 通过序号选择实例 → 输入多开实例名称（自定义），你将看到类似这样的输出：
    ```
    ✅ 多开实例创建成功！
    实例ID: 24cb4c8f
    是否立即启动这个多开实例？ [y/n]:
    ```
2. 根据提示，输入`y`回车，程序将询问你如何启动，这里与你启动第一个实例相同。
3. 启动完成后，你将看到如下输出：
    ```
    [🚀 实例多开管理]
    ==================================================
    [端口使用状态]
      MoFox_bot实例:
        - 多开实例_24cb4c8f: 主程序(8000) + NapCat(8001)

    ┏━━━━━━━━━━┳━━━━━━━━━━━━━━━━━━━┳━━━━━━━━━━━┳━━━━━━━━━┳━━━━━━━━━━━┓
    ┃ ID       ┃ 名称              ┃ 类型      ┃ 状态    ┃ 端口      ┃
    ┡━━━━━━━━━━╇━━━━━━━━━━━━━━━━━━━╇━━━━━━━━━━━╇━━━━━━━━━╇━━━━━━━━━━━┩
    │ 24cb4c8f │ 多开实例_24cb4c8f │ MoFox_bot │ running │ 8000/8001 │
    └──────────┴───────────────────┴───────────┴─────────┴───────────┘

    [操作选项]
     [A] 创建新的多开实例
     [B] 启动多开实例
     [C] 停止多开实例
     [D] 删除多开实例
     [E] 查看实例详情
     [Q] 返回上级菜单
    请选择操作:  ():
    ```
4. 至此已经有两个实例同时运行了，你可以通过`[E] 查看实例详情`查看实例的运行状态。这个多开实例的运行状态与单开实例相同，你可以通过`[C] 停止多开实例`停止实例，通过`[D] 删除多开实例`删除这个实例（不是真的释放实例）。这个多开实例在被关闭后，可以重新启动。
