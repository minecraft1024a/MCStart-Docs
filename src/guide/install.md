# MCStart


## [在这里下载我们最新的软件](https://github.com/MaiCore-Start/MaiCore-Start/releases)

1. 在安装本程序前，你需要安装 <mark>Python</mark> ，不仅是因为本程序依赖 Python 运行，也是因为 MaiCore 同样依赖 Python 运行。[你可以在这里选择适合的版本进行下载安装](https://www.python.org/downloads/windows/)，推荐3.12.8版本。安装时请务必勾选 <mark>Add Python to PATH</mark> 选项！！
2. 你只需要双击安装程序并确认几个选项即可安装。安装完成后，首次启动将触发依赖库的安装，不要害怕，这是正常现象。

## 启动方式

### 正常启动

直接双击桌面上的图标就好啦~

### Windows 运行
1. Win + R 键，打开运行框
2. 输入安装时提示的快捷启动命令并回车，如`mcs415b`

### Powershell/CMD 运行
在任意命令行中输入安装时提示的快捷启动命令并回车，如`mcs415b`
