# 查看运行状态
这是本程序的进程监控功能，您可以通过此功能查看本程序及那本程序启动的实例的运行状态。


## `stop <PID>`
停止指定进程，通过进程 PID 选中进程，主程序进程无法被选中。

## `restart <PID>`
重启指定的进程，通过进程 PID 选中进程，主程序进程无法被选中。

查看指定PID的进程详情，主程序也可以被查看。

## `stopall`
停止所有进程，主程序进程无法被停止。

## `q` or `quit`
退出进程监控状态，返回主菜单。

## `Tab键`
输入不完全的指令后，命令输入栏将自动匹配输入的指令，此时按Tab键可以直接填写指令。
