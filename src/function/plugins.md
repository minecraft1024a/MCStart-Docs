# 插件管理

::: caution 注意
插件功能目前仅支持MaiBot
:::

选择这个功能选项并通过<mark>输入目标实例的序列号</mark>选定实例后，程序会自动获取麦麦插件仓库中索引的插件并加载插件列表。

### 此时你可以

1. 通过输入`1 - 10`的数字来选择插件，查看插件的详细信息，并选择是否安装该插件`[I]`。
2. 通过`/*关键词*/`的语法来搜索插件，例如`/*一闪*/`。匹配优先级：`插件名字>插件关键词>插件介绍>作者名字>插件id>开源许可`
3. 通过`[A]`切换插件的过滤逻辑。默认只显示兼容当前实例版本的插件，如果版本号为classical，则无法使用任何插件，如果版本号为main或dev，则默认只能使用兼容最新版本的插件。切换为**全部插件**后将无视版本兼容显示/匹配全部插件。
4. 通过`[P]/[N]`切换上一页/下一页。
5. 通过`[M]`进入插件管理菜单 (刷新/扫描/管理/导入)
    - `[R]`刷新插件列表，重新获取插件仓库中的插件列表。
    - `[S]`扫描本地插件，遍历当前实例plugins目录下的所有插件，并注册为该实例的插件。
    - `[M]`管理已安装的插件。
    - `[I]`导入本地插件文件夹。
    - `[U]`检查更新，检查插件是否更新。
